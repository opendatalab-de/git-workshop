Grundlagen

  1. Erstelle ein Verzeichnis und initialisiere Git

    mkdir kata cd kata git init

  2. Erstelle eine Datei gg.txt mit dem Inhalt 1.1 Die Würde des Menschen ist unantastbar.

    echo "1.1 Die Würde des Menschen ist unantastbar." > gg.txt

  3. Schau dir den Status des working trees an

    git status

    Erwartete Ausgabe: Untracked files: stgb.txt

  4. Füge die Datei dem index hinzu und schau dir erneut den Status an

    git add gg.txt git status

    Erwartete Ausgabe: Changes to be committed: stgb.txt

  5. Erstelle einen Commit mit der Beschreibung add 1.1 und schau dir erneut den Status an

    git commit -m "add gg 1.1" git status

    Erwartete Ausgabe: nothing to commit, working directory clean

  6. Erstelle eine weiteren Commit mit einer neuen Daten stgb.txt mit dem Inhalt §1 Keine Strafe ohne Gesetz und prüfe danach den Status

    echo "§1 Keine Strafe ohne Gesetz" > stgb.txt git add stgb.txt git commit -m "add stgb §1" git status

    Erwartete Ausgabe: nothing to commit, working directory clean

Anzeigen von Änderungen und der Historie

  1. Füge der Datei gg.txt eine weitere Zeile hinzu: "9.1 Alle Deutschen haben das Recht, Vereine und Gesellschaften zu bilden." und lass dir von Git die Änderung anzeigen

    echo "Alle Deutschen haben das Recht, Vereine und Gesellschaften zu bilden." >> gg.txt git diff

  2. Erstelle einen Commit für die Änderung in der gg.txt

    git commit -a -m "add gg 9.1"

  3. Lass dir die Änderungen des aktuellen Commits anzeigen

    git show

  4. Lass dir alle bisherigen Commits anzeigen

    git log

    Erwartete Ausgabe: 3 Commits

  5. Lass dir alle bisherigen Commits anzeigen, die die Datei grundgesetz.txt betreffen

    git log gg.txt

    Erwartete Ausgabe: 2 Commits

  6. Benenne die Datei gg.txt in grundgesetz.txt mit Git um und erstelle dafür einen Commit

    git mv gg.txt grundgesetz.txt git commit -m "rename gg.txt"

  7. Lass dir alle bisherigen Commits anzeigen, die die Datei grundgesetz.txt betreffen

    git log grundgesetz.txt

    Erwartete Ausgabe: 3 Commits

  8. Lass dir die Änderungen des ersten Commits anzeigen

    git show [HASH] oder auch git show HEAD^^^

Branching & Merging

Zusammenarbeiten

  1. Erstelle einen Kopie mit der Option "bare" in einem neuen Verzeichnis (außerhalb des aktuellen Repositories) namens "remotekata"

    cd .. git clone --bare kata remotekata

  2. Erstelle einen Kopie in einem neuen Verzeichnis (außerhalb des aktuellen Repositories) namens "clonekata"

    git clone kata clonekata

  3. Füge im "kata" Repository das Repository "remotekata" als Remote-Repository "origin" ein

    git remote add origin ../remotekata

  4. Füge im "kata" Repository eine Datei bgb.txt mit dem Inhalt "1 Beginn der Rechtsfähigkeit" hinzu und erstelle einen Commit

    cd kata echo "1 Beginn der Rechtsfähigkeit" > bgb.txt git add bgb.txt git commit -m "add bgb §1"

  5. Übertrage den Commit ans Remote-Repository

    git push origin master

  6. Wechsle ins "clonekata" Repository und übernehme den letzten Commit

    cd ../clonekata git pull

  • Erstelle einen Branch mit dem Namen kleine_revolution

    git branch kleine_revolution

  • Lass dir alle Branches anzeigen

    git branch

    kleine_revolution * master

  • Wechsle in den Branch kleine_revolution

    git checkout kleine_revolution

    Switched to branch 'kleine_revolution'

  • Ändere die Datei stgb.txt und ersetze den Inhalt "Der König hat immer Recht"

    echo "Der König hat immer Recht" > stgb.txt

  • Erstelle einen Commit für die Änderung der stgb.txt

    git commit -a -m "liva la revolution"

    [kleine_revolution b0c6103] liva la revolution
    1 file changed, 1 insertion(+), 1 deletion(-)

  • Wechsle in den Branch master

    git checkout master

    Switched to branch 'master'

  • Füge "§ 14 Handeln für einen anderen" zu stgb.txt hinzu

    echo "§ 14 Handeln für einen anderen" >> stgb.txt

  • Erstelle einen Commit für die Änderung der stgb.txt

    git commit -a -m "neues gesetz"

    [master e22a053] neues gesetz
    1 file changed, 1 insertion(+)

  • Lass dir von Git die Änderungen zum "kleine_revolution" Branch anzeigen

    git diff kleine_revolution

    [master e22a053] neues gesetz
    1 file changed, 1 insertion(+)

  • Lass dir von Git die Änderungen zum "kleine_revolution" Branch anzeigen

    git diff kleine_revolution

    [master e22a053] neues gesetz
    1 file changed, 1 insertion(+)

  • Merge den kleine_revolution Branch

    git merge kleine_revolution

    Auto-merging stgb.txt
    CONFLICT (content): Merge conflict in stgb.txt
    Automatic merge failed; fix conflicts and then commit the result.

  • Behebe den Konflikt von Hand

    start stgb.txt oder open stgb.txt

  • Füge die gemergte Datei wieder zu Git

    git add stgb.txt

  • Erstelle einen Commit für die Änderung der stgb.txt

    git commit -a -m "final"

    [master e22a053] neues gesetz
    1 file changed, 1 insertion(+)
